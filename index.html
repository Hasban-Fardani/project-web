Tugas Git & Github

By: Hasban Fardani

Apa itu Git?

  Git adalah sebuah software VCS (Version Control System) yang berfungsi untuk memanage dan mengelola perubahan source code atau yang bisa disebut dengan SCM (Source Code Management).

  Dengan menggunakan git kita bisa melihat track perubahan file , melakukan pengembalian file ke versi sebelumnya dan bahkan bisa untuk bekerja sama dengan orang lain untuk membuat suatu aplikasi

Apa itu Github?

  Github adalah sebuah layanan penyimpanan dan pengelolaan project berbasis cloud yang memperingan penyimpanan komputer kita.

  Github harus menggunakan git untuk mengupload dan mengelola file

Apa perbedaan Git dan Github?

  Git adalah sebuah software VCS sedangkan github adalah layanan cloud untuk menyimpan source code yang dikelola menggunakan git
